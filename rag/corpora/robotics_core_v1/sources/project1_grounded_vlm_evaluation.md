# 项目一 Grounded VLM 风险感知事实卡

## 两层职责

GroundingDINO 使用开放词汇提示定位水坑候选；颜色占比、RGB-D 深度和物理尺寸规则执行确定性门控。
Qwen2.5-VL 7B AWQ 只对已接受的候选解释移动机器人策略，不替代候选检测，也不直接控制机器人。

## 负样本与无调用

候选 verified score 未达到门槛时，级联应在 Grounding gate 停止，不调用 Qwen。这样能把明显负样本
挡在低成本阶段，并避免让 VLM 对整幅图自由猜测不存在的风险。

## 冻结评测边界

rbot 水坑策略集包含 5 个正样本和 3 个负样本：策略模式得到 TP=5、FP=0、TN=3、FN=0；另有 5 个
hard negative 均在 Grounding gate 被拒绝，Qwen 调用为 0。这个小型仿真集证明数据链可复算，不能
外推成真实世界水坑识别率 100%。

## 三维投影

现场观测使用同帧 RGB-D、相机内参和观测时间戳 TF，把候选区域投影到 `map`。若历史 TF 不可用应
fail closed；不能无记录地改用最新 TF。持久化语义位置再驱动动态 Keepout，而不是直接发布速度。
